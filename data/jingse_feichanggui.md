# 【儋州龙门激浪】

内容：龙门激浪，位于儋州市峨蔓镇海滨，这里风光秀丽，火山岩地貌怪石嶙峋，海水还十分清澈，水质非常好。人们都说海南岛的日落属西部最美，而西部最美的日落当属"龙门激浪"了。儋州火山海岸"龙门激浪"经世界旅游组织专家组考察后认为是海南省滨海莽原带之一，有较高的旅游开发价值。龙门火山海岸，由密集的火山口群与熔岩流、石块构成，是显著的海蚀地貌。由于风、海水以及漫长时光的作用，目前这里显现出了海岬角和海湾。与海南岛东部海湾的椰风海韵相比，峨蔓湾有一种气势磅礴的雄壮之美。峨蔓湾除了有沙滩，还有绵延数公里的火山岩，特别是由红壤堆积成的几十米高的红色悬崖，犹如一堵高大的城墙屹立在海边，守护着这处美丽的海湾。红色的悬崖和清澈的海水形成了独特的"一半是海水一半是火焰"的自然景观。在峨蔓湾，有一座布满石头的山，名为龙门山。位于龙门山东侧的"龙门激浪"景观是儋州古"八景"之一。龙门是一处自然形成的中空岩洞，高30多米，宽20多米，像高大的拱形城门，因此也曾被称为"南天第一门"。在《儋县志》这样描述着：大海潮来，触石澎湃，浪花卷石，如万马奔腾，海风吹浪，触石而鸣，回响十余里，临海有一石，坚石如烟囱，顶平色红，谓之红花印，又曰犁头石。古时有诗曰：浪来难激红花印，差幸犁头不再生……傍晚时分的龙门激浪景区，霞光满天，市民游客来这里打卡游玩，可以吹吹海风、看看夕阳，在周边吃吃小海鲜，十分惬意。游客在峨蔓湾，不仅可以享受和煦的阳光、洁净的沙滩和湛蓝的海水，还可以观赏日落美景。夕阳西下，天空被"烧"得通红，海面金光闪闪，好像镀了一层金，景色非常壮观。利用风力发电的一个个大风车耸立在海岸边，成为峨蔓湾又一道独特的风景线。 开放时间：全天24小时 票价： 电话： 地址：海南省儋州市峨蔓镇海滨龙门山 交通： 


# 【罗帅驿站】

内容：【罗帅驿站】由于罗帅驿站位于鹦哥岭脚下，通过弯弯绕绕的山路才能抵达，一路上，你就可以尽情地享用车窗外的原始雨林美景，呼吸负氧离子极高的空气，体验罗帅人进出县城的艰辛。 罗帅驿站很有意思，它的构建完全会颠覆你对美丽乡村的看法。到达罗帅驿站，只见20多栋黄白相间的联体别墅并排着，犹如一个个考究的蜂房，门上的帖图还有很多，如一枝花、一个笑脸、一只麦克、五角星等，这可不是乱刻的。有花有笑脸的房子是可对外接待客人的，有麦克风的房子是可唱卡拉OK的，门上还有评星标准，相当于“星级酒店”的标志！ 居住在这里，清晨，推开窗子，山区特有的湿润气息便扑面而来。氤氲的雾气混着农家炊烟淡淡地浸润着人的呼吸，让人心肺都沐浴在自然的润泽中！远山躲在云雾中安睡，早起的雄鸡不时的呼唤着贪睡的山水和偷懒的人儿…… 驿站内的美景也是享用不完的，随处可见花园和菜地，可以和主人商量，自己到田地内采摘蔬菜，体验农家乐趣，如果幸运，还可以观看到黎家人原始的酿酒工艺。 而在驿站外，还可以爬山和溯溪，由于鹦哥岭是纯粹的原始热带雨林，这里野草齐腰高，攀爬过程中还可能要钻入密林，需要手脚并用地在灌木丛里钻来钻去，雨林的空气湿度很大，探险原始热带雨林的后果有可能是手脚都有伤痕，但有时候原本还远在山谷另一边的木棉花，突然红艳艳地开在了你的咫尺视野里，惊叹的感觉不言而喻。 驿站不远处就可以溯溪，因为落差的关系，有些地方水流很湍急，要很小心以防被冲倒。河床上躺着很多被溪水冲刷得光洁的石头，有些在水底有些露出了水面，小的形如鹅卵，大的有好几米的直径，不仅有石头，这儿还有瀑布可以观赏。如果时间充裕，这儿还可以骑马，体验车马奔腾的快感。一个小小的驿站，就可以满足观赏美景和溯溪探险的旅游要求。 开放时间： 票价：免费 电话： 地址： 交通：自驾：从白沙黎族自治县县城驾车沿S310向东南方向行驶18公里左右，右前方转弯，即可达到元门乡罗帅村。汽车：由海口乘坐到达白沙的汽车，再由白沙县城乘坐去往罗帅驿站的公交车。 


# 【盐丁村】

内容：【盐丁村】 这是一个用石头串起来的村庄，一个用黑色火山岩堆砌成的村落。它们一个挨一个地排列在儋州北部沿海的海岸线上，在海浪的冲刷下存在，在海盐的浸泡中成长……这就是海南儋州的盐丁村，听着是不是很有意思，那就速来感受这个有着千年历史的盐丁村。盐丁村是儋州市峨蔓镇最边远的一个村委会，海边有距今一千两百多年、总面积达两千多亩的古盐田，一千三百多个由火山岩制成的盐槽散落其中，它们大小不一、形态各异；盐丁村还有土地庙、细沙古灯塔、敬字亭、外翰林民居等文化古迹，这些文化古迹基本都是由天然的玄武岩直接雕刻而成，做工精致，工艺高超。盐丁村是典型的火山玄武岩地貌。来到这里，就可以看到星罗棋布的盐槽，有墨砚式的，有棋盘样的，还有一间间用来存放盐巴的小石屋，这些小房子，或孤立一隅，或三五成排，非常有趣。 每年一到阳光强烈的夏季，盐丁村的晒盐工作就开始了，老老小小们也开始忙碌起来。如果夏季你来到这里，就可以看到传统的晒盐工艺。盐丁村古盐田制盐工序古老、原始、独特，在制盐过程中，盐工们先利用海水与盐田的泥土混合，然后用木耙把盐泥纵横交错地耙松，晒干，待水份蒸发，盐泥的盐分升高后，把盐泥耙成一堆，放进铺有稻草、树枝的过滤池里，用脚踏实浇上海水过滤，制成盐卤水，再把盐卤水浇灌到火山岩制成的盐田上，经过日晒制成白花花的盐。盐丁村附近的沿海有大片的红树林，整个盐丁村就是一幅画，具有独特的艺术魅力。盐丁村比洋浦古盐田更原始古朴，规模更宏大，火山特色更明显。沿途还将领略海边大风车、游览著名景点“龙门激浪”、参观美丽乡村“打铁村”。如果有足够的探险精神，还可以再附近的景点“龙门激浪”野营，那的海货比较多，晚上还可以下海捕捉螃蟹。 开放时间： 票价：免费 电话： 地址： 交通：自驾：从海南岛西线高速洋浦路口下来10公里，过木棠镇，走乡村公路，到峨蔓镇再调头向西，沿一条颠簸的土路行进约4公里，见到许多黑色的盐田，盐丁村到了。汽车：由海口乘坐汽车到达木棠镇，在乘坐风采车或者摩托车抵达。

# 【羊角岭天池旅游区】

内容：【羊角岭天池旅游区】羊角岭天池位于屯昌县城南4公里处的羊角岭顶端。羊角岭海拔200多米，为我国最大型、最富集的水晶矿床所在地，也是当今世界上超大型水晶矿床之一。羊角岭水晶矿也称为七零一矿。矿床类型属矽卡岩中英脉型，主矿体长240米，宽90-130米，延深150米。主矿体周围尚有十几个金矿小矽卡岩体和大片砂矿分布。羊角岭水晶原矿以其透明度高。质地纯净而著称。羊角岭水晶矿最早由日本人在二战期间开采，日寇对羊角岭矿区进行过掠夺性开采。自1955年起，国家进行综合勘探后正式建矿生产，是我国最早的天然水晶矿，至70年代中期主要矿体开采完毕。羊角岭巨型水晶矿，以规模大、品位高、质量好闻名于世，为我国尖端军工业以及偿还外债作出了巨大贡献。此地还是一个很好爱国主义教育基地。1939年日军进行掠夺性挖采，解放后由我国有关部门进行开采，直至现在。天池为采矿挖掘而成，宽约三十米，长约七十米，深约200米，水清澈透明，夏凉冬暖，常年有游泳爱好者到期此尽情畅游。 开放时间： 票价：免费 电话： 地址：屯昌县城南4公里处的羊角岭顶端 交通：自驾：海口出发，经丘海大道、中线高速进入X380，直行到达；三亚出发，经海南环线高速、琼乐高速，进入X380直行到达。
